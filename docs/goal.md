# Goal of djutils-build-parent

The project offers a parent pom with default settings that can be kept constant across projects. 
Projects can still override every setting.

Provides:
- Java version
- compiler settings
- test execution
- code coverage
- formatting
- Checkstyle
- SpotBugs
- PMD

Used by:
- DJUTILS
- DJUNITS
- DSOL
- OpenTrafficSim
- SIM0MQ
